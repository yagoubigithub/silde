Les technologies Web 2.0

AJAX :la technologie qui ajoute a  navigateur web  il combin javascript pour modifie le contenu de navigateur et XMLHttpRequest pour envoi les donnee vers le serveur

Application Web:est une application manipulable grâce à un navigateur web.
Dans la technologie client-serveur le navigateur Web envoie au serveur des requêtes Le serveur répond et envoie les pages au navigateur Web. Le navigateur affiche les pages à l'utilisateur.
Les applications Web utilisent cette technique pour affiche leur interface graphique	

blogs et resaux sociaux
